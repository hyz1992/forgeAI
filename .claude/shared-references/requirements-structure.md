# 需求文档结构（docs/requirements.md）

**校验要求**：须包含 **项目名称、项目目标、项目类型**，至少一个**核心功能**（含描述、优先级、验收标准），以及至少一条**整体验收标准**。内容过少（不足 100 字）会校验失败。生成后必须运行 `python .claude/scripts/validate.py requirement`，不通过则补全或修正直至通过。
